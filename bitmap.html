JPG: Joint Photographic Expert Group

  • VENTAJAS
    1. Admiten compresion
    2. Alta fidelidad y formato ligero
    3. Soportan Hasta 16.777.216 colores
  • DESVENTAJAS
    1. No admiten Transparencia
    2. Se pixelan cuando se estiran demasiado
    3. Se veran distorcionadas si la resolucion es pequeña

PNG: Portable Network Graphics

  • VENTAJAS
    1. Admiten Transparencia
    2. Alta fidelidad
    3. Soportan Hasta 16.777.216 colores
  • DESVENTAJAS
    1. No admiten compresion
    2. Unico recurso para presentaciones con transparencias
    3. Formato pesado para web

GIF: Graphics Interchange Format

  • VENTAJAS
    1. Admiten compresion
    2. Formato ligero
    3. Admiten interpolación (Animación)
  • DESVENTAJAS
    1. Admiten hasta 256 Colores
    2. Generalmente de Baja calidad y resolucion pequeña
    3. No es ideal para galerias fotograficas o grandes formatos
