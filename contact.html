Tipos de Formularios

  • Busqueda (Consulta query)
  • Inscripcion (Registro)
  • Sesion (Acceso)
  • Consulta Web (Contacto)

Los formularios son elementos que se utilizan para el intercambio de datos entre el usuario y el servidor, Por ende, podemos decir que permite una interaccion real. Dentro de un formularios encontraremos diferentes elementos, entre ellos estan los campos ( <input />, <textarea></textarea> <select></select> ) asi como tambien las etiquetas ( <label></label> ) que se utilizan para identificar el tipo de dato que debe ingresar el usuario.
