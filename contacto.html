Formularios HTML

¿Que son?

Son elementos que permiten un intercambio de datos entre el cliente y el servidor, esto se realiza para acceder a determinada informacion o almacenar datos en el servidor; Esto se conoce como una interaccion real entre el cliente y el servidor. Los formularios se componen de dos elementos principales, las etiquetas <label> que funcionan como rotulos que indican al usuario que informacion debe ingresar, y los otro lado estan los campos de formulario <input> - <textarea> - <select> que permiten al usuario seleccionar o ingresar datos, en la pagina.

¿Cuantos tipos Existen?

Segun la finalidad del mismo, podemos mencionar dos grupos grandes de formularios; por un lado, estan los que utilizan el method="GET" para obtener informacion de un servidor, permite el envio de hasta 8kb; por otro lado, los que utilizan el method="POST" para mantener la seguridad de los datos a enviarse (la informacion se encripta) y que estos sean administrados por el servidor, ya sea para ser almacenados o reutilizados de alguna manera, permite el envio de hasta 8mb.

  • Method GET
    • Buscador - Consulta Query
  • Method POST
    • Inscripcion - Registro
    • Acceso - Inicio Sesion
    • Consulta - Contacto

¿Cuantos Tipos de Campos Existen?

Los formularios admiten varios tipos de campos segun la informacion que debamos ingresar o seleccionar de los mismos, entre los elementos tenemos las listas desplegables <select> que permite marcar la opcion deseada de una lista de opciones; el campo multilinea <textarea> tambien conocido como caja de comentarios, utilizado para escribir en varias lineas el texto deseado; y los campos de una sola linea <input> que poseen diferentes tipos, segun la informacion solicitada en el formulario.

  • TEXTO (Aa1#!)
  • NUMERO (123)
  • CORREO (a@a)
  • FECHA (dd/mm/aaaa)
  • OPCION UNICA
  • OPCION MULTIPLE
  • REESTABLECER
  • ACCION PREDETERMINADA

Contenido Multimedia

Elemento Audio

Uno de las nuevas incorporaciones que incluye HTML5 es el elemento <audio> que nos permite trabajar con varios formatos multimedia. Entre los principales formatos de sonido, soporta .mp3, .wav, .ogg, aunque No todos los navegadores soportan este ultimo; asi como tambien permite la reproduccion de elementos de video cuyos formatos compatibles se veran el la seccion Elemento Video

Elemento Video

Al igual que el elemento <Audio> el elemento <video> se incorporo en HTML5, este admite los formatos .mp4, .avi, .ogv, y al igual que sucede con el formato de audio .ogg, este tampoco es compatible con todos los navegadores; tambien debemos incorporar el atributo controls para que estos elementos se visualicen en pantalla. 16:9 (ancho 1.7 > alto) 4:3 (ancho 1.3 > alto)

Contenido EMBED

El contenido embebido es una categoria que se incorporo en las ultimas versiones de HTML, es probable que muchas veces incorporemos elementos de esta categoria a travez de la etiqueta <iframe>, tambien conocida como marco flotante, permitiendonos insertar contenido externo en nuestra web de manera asincoronica. Esto permite que el contenido del marco flotante no afecte a los tiempos de carga de nuestro sitio.
