Formularios

¿Que son?

Son una herramienta que permite al usuario tener una interaccion real con el servidor, es decir, que se produce un intercambio de datos y hay una respuesta en base a la informacion ingresada. Estos se componen de 2 elementos principales: etiquetas (<label>) que permiten al usuario identificar el tipo de contenido que se debe ingresar & campos (<input><textarea><select>) que permiten al usuario el ingreso, seleccion o manipulacion de los datos.

¿Como Funcionan?

Los formularios cuentan con dos atributos principales: action el cual indica la accion predeterminada a la hora de enviar el formulario (antiguamente se utilizaba el valor action="mailto:cristiandracedo@hotmail.com", pero actualmente se implementa un script en lenguaje de servidor) & method que indica la forma en la que se van a presentar los datos, existen dos valores posibles: GET (metodo menos seguro, ya que no se encripta la informacion; ideal para buscadores ya que solicita los datos al servidor y lo ingresado en el formulario no es relevante. Admite hasta 8kb) POST (metodo seguro, encripta los datos a enviarse y estos no son accesibles desde la barra de direcciones, los datos ingresados son prioridad y admite hasta 8mb)

    Method
  • GET

  • POST

    • Acceso - (Inicio de Sesion)
    • Consulta Web - (Contacto)

¿Tipos de Datos?

En los campos de formulario, podremos ingresar o seleccionar diferentes valores, representados por las siguientes etiquetas: <select> o campo de lista desplegable, permite al usuario seleccionar un item de una lista; <textarea> o campo de texto multilinea, permite al usuario ingresar varias lineas de texto y por defecto es redimensionable; e <input> o campo de una sola linea que posee multiples tipos de datos, los cuales podemos especificar mediante el atributo type="text"
