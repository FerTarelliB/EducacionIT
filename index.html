Etiquetas de Bloque: Son aquellas que por defecto ocupan todo el ancho de la ventana, poseen ancho y alto y se pueden manipular con facilidad. Este tipo de etiquetas, no comparten la linea con otras etiquetas.
Etiquetas de Linea: Son aquellas que pueden compartir el mismo espacio (linea) con otras etiquetas de linea y no poseen ancho ni alto definido. Este tipo de etiquetas es mas dificil de manipular.

Etiqueta de Bloque que se utiliza por cuestiones de diseño, por ejemplo, para contener y seperar parrafos o imagenes. Etiqueta de Linea que se utiliza por razones esteticas, por ejemplo, cambiar los atributos de una letra, una frase o elemento que corresponda a la linea

Las etiquetas HTML cuentan con las siguientes caracteristicas:

  • Margin: Distancia que separa una etiqueta de otra. NO SUMA TAMAÑO y siempre aplica el mas grande
  • Border: Linea que rodea las etiquetas. SUMA TAMAÑO y por defecto su valor es 0
  • Padding: Distancia que separa el border del contenido. SUMA TAMAÑO y por defecto en la mayoria de los casos su valor es 0
  • Content: Elemento que se encuentra dentro de una etiqueta HTML, POSEE TAMAÑO y en caso de ser de Bloque este es todo el ancho de la ventana, caso contrario, solamente ocupa el espacio correspondiente al contenido
