Tema 1: Elementos de Medicion

En HTML5 se incorporaron dos nuevas etiquetas para mostrar informacion de manera estadistica.
Estos elementos son:

<progress> : Indica cuanto llevamos realizado de una tarea, mientras que esta aun no este finalizada. Se implementa tanto para barras de carga como para indicar un avance respecto a cualquier tipo de actividad

  • Avance: 20/100
  • Avance Indefinido Ingrese un Valor

<meter> : Indica los valores correspondientes a una actividad finalizada, como una votacion o el valor de un examen. A diferencia de progress se pueden establecer diferentes caracteristicas en base a los valores optimos y bajos.

Tema 2: Multimedia

En HTML5 se incorporaron las etiquetas de <audio> y <video> para la implementacion de contenido multimedia, sin necesidad de algun plugin y compatibilidad con multiples formatos.

<audio>: Reproduce formatos .wav, .mp3 & .ogg; asi como tambien las pistas de audio, alojadas en los archivos de video.

<video>: Admite los formatos .avi, .mp4 & .ogv. <source> se utiliza para definir varios origenes de archivos, en caso de incompatibilidad.

<iframe>: Utilizado para la insercion de contenido embebido en HTML de manera asincronica, es muy comun para la implementacion de mapas o videos almacenado en otros servidores.

Autor: Cristian Racedo

Tema 3: Elementos Graficos

HTML5 ofrece nuevas herramientas para la implementacion de graficos, estas son <svg> & <canvas>; en las que deberemos aplicar lenguajes como XML o JS, para la creacion de los diferentes elementos.

Autor: Cristian Racedo

<svg>: Scalable Vector Graphics, son imagenes generadas mediante calculos matematicos XML, ideales para graficos que deben implementarse en multiples tamaños sin que pierdan calidad.

<canvas>: Utiliza Bitmap para crear un lienzo JS. No son escalables, pero permite el desarrollo apicaciones, mediante EventListeners.

Debe tener habilitado JS para poder utilizar Canvas
