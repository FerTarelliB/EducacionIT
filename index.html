Tema 2: Elementos Estadisticos

Con la llegada de HTML5 se incorporaron 2 nuevos elementos para la utilizacion de valores estadisticos o de medicion. Estos son <progress> que se utiliza para representar el avance o la evolucion respecto a una tarea que se lleve a cabo, pero que todavia no haya finalizado; y <meter> que se utiliza para la representacion de valores estadisticos o comparativos de una tarea o actividad finalizada.

  • Elemento Progress
    • sin value
    • con value
  • Elemento Meter
    • value <= low
    • low < value <= high
    • low < high < value <=optimum

Tema 3: API's de Dibujos

Los navegadores integran dos potentes herramientas a la hora de trabajar con imagenes/dibujos, estas son <svg> que utiliza XML para crear graficos vectoriales (imagenes calculadas matematicamente, compuestas de lineas y curvas) en nuestra web; por otro lado <canvas> permite al desarrollador crear graficos mapa de bits (imagenes diagramadas utilizando una grilla, que permite la generacion de campos de profundidad con la disposicion de los puntos/pixeles) mediante la utilizacion de JS

Tema 1: Colores CSS

Existen multiples formas de utilzar color en la web, la mas sensilla es Color Names, que admite hasta 140 colores diferentes; otras utilizan la sinstesis aditiva, combinando los canales Rojo, Verde y Azul para la obtencion de tonalidades, como RGB(red,green,blue) o Hexadecimal #rrggbb, siendo la unica diferencia, la notacion de los colores. Por ultimo, se incorporo el modelo de progresion con HSL(hue°,saturation%,lightness%) donde se puede elegir un valor de color del circulo cromatico, variando entre saturacion e iluminacion del mismo.

Tema 2: Regla @font-face

Cuando trabajamos con tipografias, podemos elegir entre aquellas que suelen estar instaladas en la mayoria de los equipos, conocidas como fuentes seguras , repositorios de fuentes como Google Fonts , donde podemos seleccionar un listado de tipografias para implementar en nuestro sitio mediante la regla @import. Asi como tambien se pueden implementar fuentes de fantasia mediante la regla @font-face , donde deberemos indicar la ubicacion y el nombre que recibira la fuente que tengamos en nuestro sitio.
