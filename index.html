Tema 1: Contenido Multimedia

HTML5 incorpora dos nuevos elementos para la reproduccion de este tipo de contenido, por un lado tenemos <audio> que admite formatos como .mp3 - .wav - .ogg, aunque tambien admite formatos de video. Por otro lado el elemento <video> admite formatos .mp4 - .avi - .ogv.

Tema 2: Elementos Estadisticos

Con la llegada de HTML5 se incorporaron 2 nuevos elementos para la utilizacion de valores estadisticos o de medicion. Estos son <progress> que se utiliza para representar el avance o la evolucion respecto a una tarea que se lleve a cabo, pero que todavia no haya finalizado; y <meter> que se utiliza para la representacion de valores estadisticos o comparativos de una tarea o actividad finalizada.

  • Elemento Progress
    • sin value
    • con value
  • Elemento Meter
    • value <= low
    • low < value <= high
    • low < high < value <=optimum

Tema 3: API's de Dibujos

Los navegadores integran dos potentes herramientas a la hora de trabajar con imagenes/dibujos, estas son <svg> que utiliza XML para crear graficos vectoriales (imagenes calculadas matematicamente, compuestas de lineas y curvas) en nuestra web; por otro lado <canvas> permite al desarrollador crear graficos mapa de bits (imagenes diagramadas utilizando una grilla, que permite la generacion de campos de profundidad con la disposicion de los puntos/pixeles) mediante la utilizacion de JS
